Instalación en Windows

  1. Descargue el instalador desde la página oficial de TaskMaster.
  2. Ejecute el archivo TaskMasterInstaller.exe.
  3. Siga las instrucciones del asistente de instalación.
  4. Una vez completada la instalación, inicie TaskMaster desde el menú de inicio.

Instalación en macOS

  1. Descargue el archivo TaskMaster.dmg desde la página oficial.
  2. Abra el archivo .dmg y arrastre TaskMaster a la carpeta "Aplicaciones".
  3. Ejecute TaskMaster desde la carpeta "Aplicaciones".

Instalación en Linux

  1. Abra una terminal y ejecute el siguiente comando para descargar TaskMaster:
    2. wget https://taskmaster.com/download/taskmaster-linux.tar.gz
  2. Extraiga el archivo descargado:
    3. tar -xvzf taskmaster-linux.tar.gz
  3. Navegue a la carpeta extraída y ejecute TaskMaster:
    4. cd taskmaster
./taskmaster
