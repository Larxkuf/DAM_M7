Configuración Inicial

Después de instalar TaskMaster, sigue estos pasos para configurar la aplicación:

  1. Inicie TaskMaster y complete el asistente de configuración inicial.
  2. Configure la conexión a la base de datos (si es necesario).
  3. Personalice las preferencias de usuario, como el idioma y el tema.

Configuración de la Base de Datos

TaskMaster soporta MySQL y PostgreSQL. Para configurar la base de datos:

  1. Abra el archivo de configuración config/database.yml.
  2. Ingrese los detalles de la conexión (host, usuario, contraseña, nombre de la base de datos).
  3. Guarde los cambios y reinicie TaskMaster.

Personalización de la Interfaz

Puedes personalizar la interfaz de TaskMaster desde el menú Configuración > Apariencia.

  • Tema: Elige entre modo claro y oscuro.
  • Idioma: Selecciona el idioma de la interfaz.
  • Notificaciones: Configura cómo recibir notificaciones.
